0.1 - Reddit feedback

Release Date: Sep. 20, 2015

Changelog

  • Added a changelog (thanks to Volbric)
  • Here you can read about all the updates I make to this game (thanks to me :)

    • Small additions

  • Pokemon can now be sorted by PokeDex entry, Name, Level, Attack, and order captured (thanks to Pedter)
  • The captured pokemon lists automatically updates when you capture a new one (thanks to Hannomeggel)
  • A little pokeball it added to a wild pokemon name if you already caught it (thanks to TheTreester)

    • Bug fixes

  • Fixed a bug where switching routes while the game was in capture state to catch the newly spawned pokemon (thanks to Virgildiablo)
  • Fixed seels being everywhere!
  • Fixed money gained from selling not showing correctly in the logs (thanks to Hannomeggel)
  • Removed pokeballs until I implement a way to regain them (thanks to Yocoolman)
  • Fixed the pokemonbox clipping with the log (thanks to Rarylith)
  • Fixed the upgradebox being pushed down by the pokemonbox (thanks to Rarylith)
  • Fixed a javascript floating point error (thanks to Rarylith)
  • Upgrades no longer show without buying the previous upgrade (thanks to Quietlark)
  • Fixed a bug where you could buy PokeballUpgrade II before I (thanks to G-ice)

    • 0.09 - Legendaries

    Release Date: Sep. 18, 2015

    Legendaries

  • The player now has a small chance to encounter legendaries
    • Legendaries are rare pokemon who are a lot stronger than other pokemon
    • Articuno, Zapdos, Moltres, Mew and Mewtwo are all in the game!
    • Everytime a pokemon spawns it has a chance to be a legendary
    • This chance can of course be improved by buying upgrades

    0.08 - Dynamic catch rate

    Release Date: Sep. 16, 2015

    Pokemon

  • Catch rate is no longer fixed for all pokemon
    • Weak pokemon have a higher chance to be caught than strong pokemon
    • Your overall catch chance can still be upgraded!

    Savefile

  • You no longer need to delete your savefile every time the game updates

    • 0.07 - Evolution

    Release Date: Sep. 15, 2015

    Pokemon

  • Pokemon can now evolve!
    • When a pokemon reaches a high enough level it can evolve
    • When a pokemon evolves, you keep the pokemon and get the evolution at level 1
    • Example: I have a level 15 Pidgey, and it reaches level 16. I end up with a level 16 pidgey and a level 1 pidgeotto
